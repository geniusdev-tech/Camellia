# Roadmap

## Short Term (v2.2)
- [ ] **Stealth Mode UI**: Toggle to hide "Secure" files completely from the list unless a specific PIN is entered.
- [ ] **Session Timeout Config**: Allow user to set timeout duration (currently fixed 5 min).
- [ ] **File Shredding**: Securely overwrite original files after encryption.

## Medium Term (v3.0)
- [ ] **Virtual File System**: Mount vault as a drive (FUSE/WebDAV) instead of renaming files in place.
- [ ] **Cloud Sync**: Encrypted sync to Google Drive / OneDrive.
- [ ] **Mobile App**: React Native viewer for the vault.

## Long Term
- [ ] **Hardware Key Support**: YubiKey integration for unlocking the Master Key.
- [ ] **Audit Logs**: Tamper-evident logs of all file access.
